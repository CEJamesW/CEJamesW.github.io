录取分配说明：
┌────────────┬────────────────────────────────────────────────┐
│ 规则项     │ 说明                                           │
├────────────┼────────────────────────────────────────────────┤
│ 综合排序   │ 按综合录取分从高到低处理学生。                 │
│ 综合录取分 │ 高考成绩 + 兴趣课程匹配数 × 5。                │
│ 兴趣匹配   │ 计算学生兴趣课与可报专业关联课程的最高匹配数。 │
│ 同分规则   │ 综合分相同先看高考成绩，再按学号升序。         │
│ 志愿顺序   │ 按学生填报的大学志愿顺序依次尝试录取。         │
│ 学校容量   │ 学校满额则跳过，继续尝试下一志愿大学。         │
│ 专业录取   │ 优先录取到学生在该校选择且仍有容量的专业。     │
│ 专业调剂   │ 服从调剂时，进入该校仍有容量的其他专业。       │
│ 未录取情况 │ 本校失败则尝试下一志愿；全部失败则暂未录取。   │
│ 重复执行   │ 执行前清空旧结果，再按当前数据重新计算。       │
└────────────┴────────────────────────────────────────────────┘
